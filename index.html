    Eventos

    São ações disparadas pela interação dos usuários na página, tornando assim as páginas interativas e dinâmicas. Ele chama a função selecionada. Mais utilizados:

  • onclick - disparado quando recebe um click
  • ondbclick - clique duplo
  • onmouseover - dispara quando o mouse está sobre
  • onmouseout - disparado quando o mousen é movido para fora do elemento
  • onmousemove - o mouse é movido no elemento
  • onmousedown - clique do botão foi pressionado
  • onmouseup - clique do botão é liberado
  • onfocus - disparadp quando elemento recebe o foco. válido para input
  • onchange - existe uma mudança no conteúdo. "ao mudar"
  • onblur - o elemento perde o foco
  • onkeydown - disparado quando uma tecla é pressionada
  • onkeypress - disparado quando uma tecla é pressionada solta
  • onkeyup - um tecla é solta sobre um elemento
  • onload - disparado quando a página terminou de ser carregada. Body
  • onresize - disparado quando há um redimencionamento da janela

Array usa colchete, objeto é chaves. Array sempre começa com 0. é possivel quebrar linha deixando mais facil de identificar os itens. Pode se criar uma array vazia e inserir os dados depois, inclusive dizendo em que posição cada item deve ficar ou array preenche a medida que inserir novos dados . outra forma é a seguinte. const lista = new Array("arroz", "feijão", "macarrão", "leite"); ex.: const lista = ["arroz", "feijão", "macarrão", "leite"]; Posição 0 vai conter "arroz" Posição 3 vai conter "leite" Alterar o valor de um item da array. const lista = ["arroz", "feijão", "macarrão", "leite"]; lista[0] ="café"; alert(lista[0]); ver a lista completa da array. const lista = ["arroz", "feijão", "macarrão", "leite"]; lista[0] ="café"; console.log(lista); Em uma array pode conter outra array, função, objeto. length diz quantos itens tem numa array Ex.: const lista = ["arroz", "feijão", "macarrão", "leite"]; alert(lista.length); retornara valor 4 Caso não saiba quantos itens tem na array e queira o ultimo intem da lista. pode se utilizar alert(lista[lista.length -1]); Acresencentar item na array utiliza o metodo push. const pessoa =["Rosa", "Chagas", 38, "estudante"]; pessoa.push("brasileira") ; console.log(pessoa); ou pode se utilizar: pessoa[pessoa.length] = "solteira" para saber se é uma array é so utilizar Array.isArray(pessoa) Para remover dados da matriz usa-se o metódo: pessoa.pop(); remover ultimo item da matriz pessoa.shift(); remover o primeiro item da matriz delete.pessoa[0]; pessoa.unshift("Maria ") adciona item na primeira posição e altera o indice dos demais concat junta duas matrizes, deve ser criada uma nova variavel.
